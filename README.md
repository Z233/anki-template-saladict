# 沙拉查词 AnkiDroid 模版

## :warning: 注意

这个项目正处于开发状态，目前仍有少量 BUG，且部分 UI 与交互并未完善。

## :page_with_curl: 介绍

一个适配[沙拉查词](https://github.com/crimx/ext-saladict)的 Anki 导出格式的模板，适用于 AnkiDroid。


## :star: 特点

- 虚拟键盘
- 拼写检查
- 复习状态
